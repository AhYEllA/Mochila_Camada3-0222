a. ¿Quién inventó el sistema de control de versión Git y por qué?
b. ¿A quién pertenece actualmente Github y por qué?
c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

RESPUESTA
a.El sistema de control de versiones Git fue diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de
versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. Su propósito es llevar registro de los cambios en archivos de
computadora.
b. Actualmente Github le pertenece a Microsoft por la compra  realizada el 4 de junio de 2018 por una suma de 7500 millones de dolares.
c. Si, se puede trabajar con Github sin usar laterminal a través de la aplicación de escritorio Github.
